# FairColors - Microfinance Loan Management System

FairColors is a modern, responsive microfinance loan management dashboard built with React, TypeScript, and Tailwind CSS. It provides a comprehensive suite of tools for managing customers, loans, transactions, and system settings with full dark mode support.

## 🚀 Features

- **Comprehensive Dashboard**: Real-time overview of total disbursed loans, active customers, and recent activities.
- **Customer Management**: Detailed customer profiles, status tracking, and easy navigation.
- **Loan Lifecycle Tracking**: Manage loan applications from submission to approval, rejection, or disbursement.
- **Transaction History**: Track all financial movements including disbursements, repayments, and fees.
- **Real-time Notifications**: Stay updated with system alerts and loan status changes.
- **Advanced Settings**: Customizable user preferences, including dark mode, 2FA, and language settings.
- **Responsive Design**: Optimized for both desktop and mobile devices.
- **Dark Mode**: Full support for light and dark themes with a seamless toggle.

## 🛠️ Tech Stack

- **Frontend**: React 18, TypeScript, Vite
- **Styling**: Tailwind CSS
- **Animations**: Motion (framer-motion)
- **Icons**: Lucide React
- **UI Components**: Custom-built reusable components (Button, Card, Input, Select, Badge, Toggle, etc.)

## 📦 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/chikamso-cmd/faircolors.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

## 🎨 Design System

The project uses a custom design system built on top of Tailwind CSS, focusing on:
- **Typography**: Inter for UI, JetBrains Mono for data.
- **Color Palette**: Professional slate and primary blue accents.
- **Components**: Atomic design principles with reusable UI primitives.
